# Shot-Type-Classification

TensorFlow.js를 이용한 동영상 Shot Type 감지

# Label

각 레이블별 1000개의 이미지를 사용하여 학습

-   Close Up
-   Extreme Close Up
-   Extreme Wide Shot
-   Long Shot
-   Medium Close Up
-   Medium Shot
